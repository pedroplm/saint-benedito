# Workflow

Para toda alteração feita no projeto, seguir estritamente:

1. **Fazer as alterações necessárias**
2. **Rodar lint check**: `npm run lint`
3. **Rodar build check**: `npm run build`
4. **Se lint e build passarem**: rodar `git add .`, `git commit -m "mensagem"`, `git push`
5. **Se algo falhar**: corrigir antes de commitar
